Selected Projects

Delay-Compensated EKF/UKF/CKF Framework
Developed a unified estimation architecture that compensates for time-varying sensor delays in LiDAR, IMU, and camera measurements. Achieves increased robustness in autonomous navigation under real-world latency conditions.
Autonomous Motion Planning using MPC + Reinforcement Learning
Designed a hybrid MPC–RL framework for safe and adaptive navigation in dynamic environments. Integrated into ROS2-based platforms for real-time demonstration.
6-DOF Robotic Manipulator Control & Estimation
Developed state estimation algorithms and trajectory optimization for a 6-DOF robotic arm using Python, C++, ROS2, and real sensor data. Included quaternion-based EKF for orientation tracking.
LiDAR–IMU–Camera Multi-Sensor Fusion Module
Built a modular perception framework supporting LiDAR feature extraction, IMU integration, camera-based semantic cues, and unified probabilistic fusion.
Real-Time Autonomous Navigation Platform
Integrated ROS2, nav2, MoveIt, and embedded control systems to develop a fully autonomous mobile robot capable of mapping, localization, and obstacle avoidance.
